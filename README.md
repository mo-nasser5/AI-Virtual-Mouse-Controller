# 🖐️ Gesture Controller — دليل التثبيت والتشغيل

## المتطلبات
- Python 3.8 أو أحدث (مجاني من python.org)
- كاميرا ويب

---

## تثبيت المكتبات (مرة واحدة فقط)

```bash
pip install opencv-python mediapipe pyautogui numpy
```

**لتحكم في الصوت على Windows (اختياري):**
```bash
pip install pycaw comtypes
```

---

## تشغيل البرنامج

```bash
python gesture_controller.py
```

---

## دليل الإشارات

| الإشارة | التحكم |
|---------|--------|
| ☝️ سبابة فقط | تحريك الماوس |
| ✌️ سبابة + وسطى قريبتين | كليك يسار (ضم الأصابع) |
| ✌️ سبابة + وسطى مكرر | Double Click |
| 🤟 سبابة + وسطى + بنصر | كليك يمين |
| 🖐️ كل الأصابع مفتوحة | تمرير (سكرول) — حرّك يدك فوق/تحت |
| 🤙 إبهام + خنصر | تحكم في الصوت — بُعد الأصابع = مستوى الصوت |
| 🤌 وسطى + بنصر + خنصر فقط | نسخ Ctrl+C |
| 👌 إبهام + سبابة قريبتين | لصق Ctrl+V |
| ✊ قبضة كاملة | إغلاق البرنامج |

---

## نصائح للحصول على أفضل أداء

1. **الإضاءة** — استخدم إضاءة جيدة من الأمام، تجنب الضوء خلفك
2. **المسافة** — ارفع يدك على بُعد 30–60 سم من الكاميرا
3. **الخلفية** — خلفية بسيطة غير مزدحمة تعطي نتائج أفضل
4. **السرعة** — حرّك يدك ببطء في البداية حتى تتعلم الإشارات

---

## تخصيص الإعدادات

في أول الملف `gesture_controller.py` يمكنك تعديل:

```python
CLICK_THRESHOLD    = 40   # كلما قل، كلما احتجت تضم أصابعك أكثر للكليك
SMOOTH_FACTOR      = 5    # كلما زاد، حركة الماوس أسلس (لكن أبطأ)
DOUBLE_CLICK_DELAY = 0.35 # سرعة الداب كليك بالثانية
```

## المشاكل الشائعة

**الكاميرا لا تفتح:** تأكد إن كاميراك مش مستخدمة في برنامج تاني، وجرب تغيير `cv2.VideoCapture(0)` إلى `cv2.VideoCapture(1)`

**الماوس يتحرك ببطء أو بسرعة:** قلل أو زد `SMOOTH_FACTOR`

**كليك يحصل تلقائياً:** زود `CLICK_THRESHOLD` لرقم أكبر
